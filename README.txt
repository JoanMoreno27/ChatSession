Proyecto que simula un chat entre usuarios utilizando un cliente servidor de java
utilizando Sockets y Threads para la gestión del mismo.

Cuando un usuario (ChatClient) entra en la sesión este puede ejecutar comandos 
para: iniciar sesion e informar a los demás de su llegada, mandar mensajes a 
todos los usuarios del chat, mandar un mensaje encriptado de manera simétrica
a un usuario en especifífico o cerrar la sesión mandando también un mensaje a 
todos los usuarios.
